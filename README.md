# Eye Movement Exercise
MIT xPro code assignment. Eye Movement was an exercise on how mouse events can manipulate elements on the screen. In the case of this exercise, as you hover your mouse over the screen the two ball elements inside the oval elements follow the mouse cursos, mimicking the effect of being followed by a person's eyes.

# Instalation
Download the html file and run from your browser.
